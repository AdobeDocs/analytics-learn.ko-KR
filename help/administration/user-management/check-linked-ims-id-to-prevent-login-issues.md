# 로그인 문제를 방지하기 위해 연결된 IMS ID 확인

Adobe Analytics에서 연결된 IMS ID가 잘못되거나 누락되면 사용자에게 로그인 문제가 발생할 수 있습니다. 사용자가 연결된 IMS ID를 직접 확인하고 문제가 있는 경우 고객 지원 팀에 문의할 수 있으므로 일반적인 문제 해결 단계를 수행하는 번거로움을 줄일 수 있습니다.
