# 타임스탬프 선택 사항 설정 활성화

타임스탬프 옵션을 사용하면 타임스탬프가 지정된 데이터와 지정되지 않은 데이터를 동일한 글로벌 보고서 세트에서 혼합하고, 타임스탬프가 지정된 데이터를 모바일 앱에서 전역 보고서 세트에 보내고, 새 보고서 세트를 만들지 않고도 오프라인 추적을 사용하도록 앱을 업그레이드할 수 있습니다.
