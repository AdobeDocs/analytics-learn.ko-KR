# 자유 형식 테이블에 하이퍼링크 만들기

여러 차원 항목에 대한 하이퍼링크를 만들어 Analysis Workspace의 자유 형식 테이블 내에서 해당 항목을 클릭하도록 하는 방법에 대해 알아봅니다. URL 값이 있는 차원 항목에 대한 하이퍼링크, URL이 아닌 값이 있는 차원 항목에 대한 사용자 지정 URL 또는 변수를 사용하여 여러 차원 항목에 대한 동적 사용자 지정 URL을 만들 수 있습니다. 자세한 설명서를 보려면 [자유 형식 테이블에서 차원에 대한 하이퍼링크 만들기](https://experienceleague.adobe.com/en/docs/analytics/analyze/analysis-workspace/visualizations/freeform-table/freeform-table-hyperlinks)를 참조하십시오.
