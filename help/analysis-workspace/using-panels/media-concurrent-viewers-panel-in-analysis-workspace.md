# Analysis Workspace의 미디어 동시 뷰어 패널

Media Analytics 고객은 미디어 동시 뷰어 패널을 사용하여 최대 동시 시청 시간 발생 위치 또는 시청 감소 발생 위치를 파악하여 콘텐츠 품질 및 시청자 참여에 대한 귀중한 통찰력을 제공하고 볼륨/규모에 대한 문제 해결 또는 계획에 도움을 줄 수 있습니다.
