# Analysis Workspace의 맵 시각화

이 비디오에서는 Analysis Workspace에서 맵 시각화를 사용하는 방법을 보여줍니다. 맵 시각화를 사용하면 다음 중 하나를 볼 수 있습니다 [!UICONTROL 모바일 위도/경도] 또는 웹(IP 주소) 데이터를 다이내믹 맵에 대해 검색하면 위치 컨텍스트에서 고객 사용을 알 수 있습니다.
