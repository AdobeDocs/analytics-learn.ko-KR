# 시각화 유형 및 개요 {#visualization-types-in-analysis-workspace}

Analysis Workspace 프로젝트에 입력할 수 있는 다양한 종류 또는 유형의 시각화에 대해 알아보고 데이터 스토리를 전달합니다.

>[!VIDEO](https://video.tv.adobe.com/v/23994/?quality=12&learn=on)
