# 작업 공간에서 구성 요소 드롭다운 사용

구성 요소 드롭다운을 사용하면 호환되는 구성 요소 목록에서 선택하여 분석을 작성할 수 있습니다. 필요한 구성 요소를 찾을 수 없는 경우 드롭다운에서 직접 새 구성 요소를 만들 수도 있습니다. 고급 사용 사례가 있을 경우 이러한 공간은 드롭 영역으로도 계속 작동합니다.
