# Analysis Workspace의 빠른 세그먼트

“빠른 세그먼트”는 Workspace 캔버스에서 직접 간단한 세분화를 경험할 수 있습니다. 빠른 세그먼트로 사용자는 분석 워크플로에서 나가지 않고 최대 세 가지 규칙으로 접촉식 세그먼트를 작성할 수 있습니다.
