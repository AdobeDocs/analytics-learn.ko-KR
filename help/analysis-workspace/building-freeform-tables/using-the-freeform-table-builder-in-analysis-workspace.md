# Analysis Workspace에서 [!DNL Freeform Table Builder] 사용

[!DNL Freeform Table Builder] 많은 차원, 분류, 지표 및 세그먼트를 끌어다 놓아 보다 복잡한 비즈니스 질문에 대한 답변을 제공하는 표를 작성할 수 있습니다. 데이터는 즉시 업데이트되지 않습니다. 대신 &quot;빌드&quot;를 클릭하면 업데이트되므로 테이블이 한 번만 업데이트되므로 시간을 절약할 수 있습니다.
