# 왼쪽 레일에서 필터(태그) 사용

필터(또는 태그)는 Analysis Workspace의 왼쪽 레일을 필터링하여 필요한 [!UICONTROL 구성 요소]를 찾을 수 있는 빠르고 간편한 방법입니다. Adobe에서 제공하는 획기적인 태그를 활용하거나 사용자 정의 태그를 추가하여 관련 [!UICONTROL 구성 요소]를 구성할 수 있습니다.
